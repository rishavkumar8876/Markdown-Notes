# 📝 Markdown Notes App

A full-stack Markdown Notes application with live preview and auto-save.

---

## 🚀 Features

- Create, edit, delete notes
- Live Markdown preview
- Auto-save functionality
- Clean UI with sidebar navigation

---

## 🛠 Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: SQLite
